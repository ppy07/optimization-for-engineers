# Optimization for Engineers

This repository contains my lab assignments for the course **Optimization for Engineers** by **Dr. Johannes Hild**.  
The project covers the implementation of core optimization algorithms in Python, including Newton methods, conjugate gradient solvers, line search techniques, constrained optimization, least-squares methods, and implicit filtering for noisy objectives.

## Implemented Topics

### LAB01 – Newton Methods and Linear Solvers
- Implemented a **Preconditioned Conjugate Gradient (PCG) solver**
- Implemented **Newton Descent**
- Tested correctness with `Check01.py`

### LAB02 – Globalized Newton Methods
- Implemented **Wolfe-Powell Line Search**
- Implemented **Inexact Newton-CG**
- Tested correctness with `Check02.py`

### LAB03 – Box-Constrained Optimization
- Implemented **Projected Backtracking Search**
- Implemented **Projected BFGS Descent**
- Tested correctness with `Check03.py`

### LAB04 – Least-Squares Optimization
- Implemented **Least-Squares Feasible Point formulation**
- Implemented **Levenberg-Marquardt Descent**
- Tested correctness with `Check04.py`

### LAB05 – Noisy Optimization
- Implemented **SUCS Gradient approximation**
- Implemented **Implicit Filtering**
- Tested correctness with `Check05.py`
